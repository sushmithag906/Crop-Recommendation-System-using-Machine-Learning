# Crop-Recommendation-System-using-Machine-Learning
Developed a Crop Recommendation System using Machine Learning to predict optimal crops based on soil nutrients and environmental factors. Implemented and compared multiple algorithms. Deployed the model using a Flask-based web application.

⚙️ Technologies Used
* Python
   - Pandas, NumPy
   - Scikit-learn
* Machine Learning Algorithms
   - Random Forest
   - KNN
   - Decision Tree
   - SVM
   - Logistic Regression
   - Naive Bayes
   - Gradient Boosting
   - Extra Trees
* Flask
   - HTML, CSS

🚀 How to Run the Project
1. Open VS Code and open the project folder
2. Install dependencies:
   "pip install pandas numpy scikit-learn flask"
3. Train the model:
   python train_and_save_model.py
4. Run the Flask app:
   "python app.py" - type this command in terminal,check the path before running
5. Open in browser:
   http://127.0.0.1:5000/

🎯 Features
